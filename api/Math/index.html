approximately​

</>
Math.approximately(
a: number,
b: number
) → boolean

Compares two floating point values if they are similar

If a or b is zero, compare that the other is less or equal to epsilon. If neither a or b are 0, then find an epsilon that is good for comparing numbers at the maximum magnitude of a and b. Floating points have about 7 significant digits, so 1.000001 can be represented while 1.0000001 is rounded to zero, thus we could use an epsilon of 0.000001 for comparing values close to 1. We multiply this epsilon by the biggest magnitude of a and b..
